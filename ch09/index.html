Examples

A simple module
A simple module for brief notification messages. The example consists of a notification that displays a message to the user, styled with a bit of color and a border to capture the user's attention. Click on the following link to view the Notification Module example .
Button module
A Button module with variants for large and small sizes, as well as color options. This way, color can be used to convey visual meaning to the user. For example, green will indicate a positive action, such as saving or submitting a form. Red will indicate a warning to help prevent the user from accidently clicking a cancel button. Click on the following link to view the Button Module Example .
Media module
A more complex module. This will be a media object consisting of four elements: div, image, body, and body-title. Click on the following link to view the Media Module Example.
